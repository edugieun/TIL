## IP, Port, Socket

IP, Port, Socket의 개념을 굳이 현실세계와 비교하자면 아래처럼 비교할 수 있다.

### 은행으로 따지면...

- IP : 신한은행
- Port : 유성구 노은점
- Socket : 3번 창구

### 아파트로 따지면...

- IP : 유성숲 오투그란데 3차
- Port : 503동
- Socket : 101호
