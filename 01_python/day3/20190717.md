- Palindrome은 앞에서부터 읽었을 때와 뒤에서부터 읽었을 때 같은 단어를 뜻한다. 따라서, 'a', 'nan', '토마토' 모두 palindrome에 해당한다.
- 단어를 입력받아 palindrome을 검증하고 True나 False를 리턴하는 함수 palindrome(word)를 만들어보아라.

```python
# 방법 1
def palindrome(word):
    
    # 문자열을 앞과 뒤를 비교하는 코드
    for i in range(0,len(word)):
        if word[i] != word[-(i+1)]:
            return False
        else:
            return True

word = input()
palindrome(word)
```
